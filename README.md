## Description

This a NestJS Restful-Api that contains a simple CRUD for movies

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# watch mode
$ yarn start

# development
$ yarn start:dev

# production mode
$ yarn start:prod
```
